PEBakery is backward compatible with WinBuilder 082.

## Build

To compile PEBakery from source, Visual Studio and .Net Framework is required.

### Recommended

- Visual Studio 2017 Community

### Requirement

- C# 7.0 Compiler
- .Net Framework 4.7.1 Developer Pack
